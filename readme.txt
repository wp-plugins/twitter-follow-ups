Extend the Follow-up Emails plugin for WooCommerce by adding the ability to Tweet your customers.

== Description ==

Please note: You must already have [Follow-up Emails](http://www.woothemes.com/products/follow-up-emails/ "Follow-up Emails") installed for this plugin to work.

== Installation ==

1. Upload the entire directory `twitter-follow-ups` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
